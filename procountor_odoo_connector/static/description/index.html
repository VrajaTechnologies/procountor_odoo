​  Import Product process


​ Import Product from Shopify to Odoo (You can import product through date-range or Product IDs)


​ Import Product process will create a Product Queue to Import All Products from Shopify


​ To process particular queue manually have to click on 'Process Queue Manually' button


​ To process all queues automatically we have given Scheduled Action


​  Update Product process


​ Update Product from Odoo to Shopify


​  Export Product process


​ Export Product from Odoo to Shopify (This process will add product into Product Listing)


​ Export Product from Odoo to Shopify (This process will export product to Shopify)


​  Import Product Inventory/Stock process


​ Import Inventory/Stock from Shopify to Odoo


​  Export Product Inventory/Stock process


​ Export Inventory/Stock from Odoo to Shopify (This scheduled action will prepare export stock data and will create inventory queue)


​ To process particular queue manually have to click on 'Process Queue Manually to Export Stock' button


​ To process all queues automatically we have given Scheduled Action


​  Import Customer process


​ Import Customer from Shopify to Odoo.


​ Import Customer process will create a Customer Queue to Import All Customers from Shopify


​  Import Order process


​ Import Order from Shopify to Odoo (You can import order through date-range or Order IDs)


​ Import Order (We have also given scheduled action to import unshipped order & create order queue)


​ This scheduled action will import cancel order & create order queue


​ Import Order process will create a Order Queue to Import All Orders from Shopify


​ To process particular queue manually have to click on 'Process Queue Manually' button


​ To process all queues automatically we have given Scheduled Action


​  Update order status process from Odoo to Shopify


​ Manually update order status from Delivery Order


​ For automatically update order status we given scheduled action
